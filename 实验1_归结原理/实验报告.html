一、实验题目

命题逻辑与一阶逻辑的归结推理实现

二、实验内容

1. 算法原理

1.1 归结推理原理

归结推理（Resolution）是一种基于反证的自动推理方法，由Robinson于1965年提出。其基本思想是：

  • 将待证明的结论取反，加入到知识库中
  • 通过不断应用归结规则，推导出矛盾（空子句）
  • 如果推导出空子句，则原结论成立

归结规则：

给定两个子句 C₁ = L ∨ C₁' 和 C₂ = ¬L ∨ C₂'，可以归结得到新子句 C = C₁' ∨ C₂'

1.2 最一般合一（MGU）

对于一阶逻辑，需要处理变量替换。MGU（Most General Unifier）算法用于找到两个表达式的最一般合一替换。

合一规则：

  • 变量可以替换为任意项
  • 常量只能与相同的常量合一
  • 函数符号必须相同，且参数可以递归合一

1.3 算法流程

1. 将知识库转换为子句集
2. 将待证明结论的否定加入子句集
3. while 未找到空子句:
   a. 选择两个子句进行归结
   b. 如果是命题逻辑，直接归结
   c. 如果是一阶逻辑，先进行MGU合一，再归结
   d. 将新子句加入子句集
4. 如果找到空子句，证明成功；否则失败

2. 关键代码展示

2.1 归结推理主函数

def ResolutionProp(KB):
    """
    实现一阶逻辑的归结推理
    :param KB: 子句集，每个子句是一个元组，包含原子命题或其否定
    :return: 归结推理的过程，每个步骤保存为字符串，按顺序存到列表中返回
    """
    steps = []
    kb_list = list(KB)
    
    # 添加原始子句
    for i, clause in enumerate(kb_list, 1):
        clause_str = str(clause).replace('\'', '').replace(' ', '')
        steps.append(f"{i} {clause_str},")
    
    # 过滤重言式
    def is_tautology(clause):
        literals = set()
        for lit in clause:
            if lit.startswith('~'):
                if lit[1:] in literals:
                    return True
                literals.add(lit)
            else:
                if '~' + lit in literals:
                    return True
                literals.add(lit)
        return False
    
    clauses = [clause for clause in kb_list if not is_tautology(clause)]
    step_num = len(KB) + 1
    found_empty = False
    tried_pairs = set()
    
    while not found_empty:
        new_clause_generated = False
        
        for i in range(len(clauses)):
            for j in range(len(clauses)):
                if i == j:
                    continue
                
                pair_key = (i, j)
                if pair_key in tried_pairs:
                    continue
                tried_pairs.add(pair_key)
                
                resolved = resolve(clauses[i], clauses[j])
                
                if resolved is not None:
                    new_clause = resolved[0]
                    used_clauses = resolved[1]
                    substitution = resolved[2]
                    
                    if new_clause == ():
                        found_empty = True
                        new_clause_str = "[]"
                    else:
                        new_clause_str = str(new_clause).replace('\'', '').replace(' ', '')
                    
                    # 格式化替换
                    subst_str = ""
                    if substitution:
                        sorted_subst = sorted(substitution.items(), key=lambda x: x[0])
                        subst_parts = [f"{var}={sub}" for var, sub in sorted_subst]
                        subst_str = f"[{','.join(subst_parts)}]"
                    
                    # 格式化归结步骤
                    if len(clauses[j]) > 1:
                        if subst_str:
                            step_str = f"{step_num} R[{i+1},{j+1}{chr(97 + used_clauses[1])}]{subst_str} = {new_clause_str}"
                        else:
                            step_str = f"{step_num} R[{i+1},{j+1}{chr(97 + used_clauses[1])}] = {new_clause_str}"
                    else:
                        if subst_str:
                            step_str = f"{step_num} R[{i+1},{j+1}]{subst_str} = {new_clause_str}"
                        else:
                            step_str = f"{step_num} R[{i+1},{j+1}] = {new_clause_str}"
                    
                    steps.append(step_str)
                    
                    if not is_tautology(new_clause) and new_clause not in clauses:
                        clauses.append(new_clause)
                    
                    step_num += 1
                    new_clause_generated = True
                    
                    if found_empty:
                        break
            if found_empty:
                break
        
        if not new_clause_generated:
            break
    
    return steps

2.2 MGU算法

def MGU(expr1, expr2):
    """
    实现最一般合一（Most General Unifier）算法
    :param expr1: 第一个原子公式
    :param expr2: 第二个原子公式
    :return: 合一替换字典
    """
    pred1, args1 = parse_expr(expr1)
    pred2, args2 = parse_expr(expr2)
    
    if pred1 != pred2:
        return {}
    
    if len(args1) != len(args2):
        return {}
    
    subst = {}
    
    for arg1, arg2 in zip(args1, args2):
        arg1 = apply_subst_to_term(arg1, subst)
        arg2 = apply_subst_to_term(arg2, subst)
        
        new_subst = unify_terms(arg1, arg2)
        
        if new_subst is None:
            return {}
        
        subst.update(new_subst)
    
    return subst

3. 创新点&优化

3.1 重言式检测

  • 在归结过程中自动检测并过滤重言式（包含互补文字的子句）
  • 避免生成无用的重言式子句，提高推理效率

3.2 重复子句检测

  • 记录已经尝试过的子句对，避免重复归结
  • 检查新生成的子句是否已经存在，避免冗余

3.3 无限循环保护

  • 设置最大迭代次数限制，防止特殊情况下的无限循环
  • 对于无法合一的情况，及时返回空字典

3. ~Student(x) ∨ HardWorker(x)

输出：

3 (~Student(x),HardWorker(x)),

6 R[2,1][x=sue] = (Student(sue),)

8 R[3,6][x=sue] = (HardWorker(sue),)
...
17 R[5,7][x=sue] = []

结果： ✓ 成功推导出空子句，证明 HardWorker(sue) 成立
